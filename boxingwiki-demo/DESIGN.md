# BoxingWiki Design System

## Brand
- **Name:** BoxingWiki
- **Tagline:** The free, interactive boxing encyclopedia
- **Style:** Editorial Brutalism — dark, high-contrast, athletic

## Colors
| Token | Hex |
|---|---|
| Background | `#0a0a0a` |
| Surface | `#141414` |
| Surface Hover | `#1f1f1f` |
| Border | `#2a2a2a` |
| Primary (Aggressive Red) | `#ff3333` |
| Primary Hover | `#ff4d4d` |
| Cyan Accent | `#00e5ff` |
| Text | `#ffffff` |
| Text Muted | `#b0b0b0` |
| Success | `#22c55e` |
| Warning | `#ffd700` |

## Typography
- **Body:** Inter
- **Display:** Barlow Condensed, uppercase, letter-spacing 0.02em
- **Sizes:** Hero clamp(2rem, 5vw, 3.5rem), Title clamp(1.5rem, 4vw, 2.5rem)

## Corners
- sm: 4px, md: 8px, lg: 16px, full: 9999px

## Depth
- Flat base with glassmorphism panels (rgba(20,20,20,0.6) + backdrop-filter blur(12px))

## What NOT to Do
- No light backgrounds
- No rounded/soft aesthetics — keep edges sharp
- No pastel colors
- No serif fonts
